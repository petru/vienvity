* Installation instructions

1) download the theme archive
2) unzip the 'vienvity' folder
3) upload the 'vienvity' folder to the /wp-content/themes/ directory on your web server
4) activate the theme in the Dashboard -> Design menu

* Customization
Several CSS files are available, providing basic customization for this theme:

1) css/type-boook.css - uses book-style fonts
2) css/white.css - white skin, loaded by default
3) css/colors.css - dark skin, loaded by default, but overridden by css/white.css
4) css/indentation.css - centers title, provides paragraph indentation
5) css/white-dark.css - darkens the text when using the white skin

To enable/disable these stylesheets: edit style.css and comment or uncomment the coresponding lines

Uncommented (active) line: @import "css/white.css";  
Commented (inactive) line: /* @import "css/indentation.css"; */
